What is the difference between an ID and a Class?

An ID uniquely identifies a single HTML element and is used once per page, while a class can be applied to multiple elements to share the same styling or behavior.

What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

The Div tag is a non-semantic container used for layout, while header, main, and footer are semantic tags that give meaning to the structure and improve accessibility.
