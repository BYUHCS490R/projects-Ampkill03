Questions

What kind of photos are good to encode using JPEG?

JPEG is good for photos with lots of colors, like pictures of people, nature, or anything with smooth color changes.

What kind of photos are good to encode using GIF?

GIF is good for simple pictures with few colors, like cartoons, logos, or small animations.

What is unique about the PNG format?

PNG keeps the picture clear and sharp without losing quality and can have see-through parts

How do you resize your photo to the size it should be displayed?

You use a program like photoshop or photos to make the picture smaller or bigger by changing its actual size.

Why should you resize photos using photo editing software instead of resizing it using CSS?

Because changing the size in a program makes the file smaller and faster to load, but resizing with CSS only changes how it looks on screen without making the file smaller.
