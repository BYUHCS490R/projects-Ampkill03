Internet

Web

It's a part of the Internet that has websites and web pages you view in a browser.

How are the Internet and Web different?

The internet are the wires and computers connected but the web are the websites a pages that you visit

ASCII

It's a way to represent basic English characters using numbers but only has 128 characters.

Unicode

It's a system that includes all characters from all languages.

What is UTF-8

it's a way to store Unicode characters using 1 to 4 bytes

URL

It's the address of a web page.

Domain Name

It's the human-readable name or easy-to-remember name used to identify a website, which maps to an IP address.

HTTP

The rules used to load web pages.It also defines how messages are formatted and transmitted.

HTTPS

It's a secure version of HTTP that uses encryption (SSL/TLS) to protect data transferred between the user's browser and the website server to keep your data safe.

IP Address

It's a unique numerical label assigned to each device connected to a computer network.

DNS

It's the system that translates or changes domain names into IP addresses so that browsers and websites can load them

W3C Recommendations

These are rules published by the World Wide Web Consortium to keep the web working smoothly

XML

A way to share and store data

XHTML

It's a version of html that folow the rules of XML

HTML

It's the standard language used to create web pages

HTML5

This is the latest version of html with new features for videos and sounds for better page structures
