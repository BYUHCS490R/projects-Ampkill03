Why is the viewport meta tag important?

The viewport meta tag helps make your website look good on mobile devices. It tells the browser how to scale the page so it fits the screen, making the content easy to read without zooming in.

More Info

How can you adjust the CSS for printing?

You can change how a page looks when printed using CSS media queries for print. This lets you hide unnecessary elements (like ads) and adjust the design for better readability.

More Info

When would you want to adjust the CSS for printing?

When you want to remove unnecessary parts (like menus or ads) for a cleaner printout.

When you want to make the text easier to read on paper.

When you want to save ink (e.g., by removing backgrounds or images).

More Info

Why do we need a viewport meta tag?

The viewport meta tag makes a webpage fit and display correctly on all screen sizes, especially mobile devices.

When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

When the same tag is styled twice, the last CSS rule written takes priority.

How do you change the CSS for a web page displayed on a phone?

You change CSS for phones by using a media query that applies styles only on small screens.
