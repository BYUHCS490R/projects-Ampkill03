What is JavaScript? How did it get its name?

  • JavaScript is a programming language used to make websites interactive. It was first called LiveScript, but it was renamed JavaScript to take advantage of the popularity of the Java language at the time.

What are the two ways you can run JavaScript?

  • In the browser (like Chrome or Firefox)
  • On the server using Node.js

What was the most popular front-end JS library?

  • jQuery

What is the most popular front-end JS library today?

  • React
